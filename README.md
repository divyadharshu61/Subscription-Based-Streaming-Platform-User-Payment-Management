Subscription-Based-Streaming-Platform

Project Overview
The Subscription-Based Streaming Platform is a Java-based web application developed using JDBC, Servlets, and Oracle Database. It manages user subscriptions and payment processing for an online streaming service. The system allows users to register, choose subscription plans, make payments, and manage their subscription status. Administrators can manage plans, monitor users, and track payments. The project follows a layered architecture (MVC pattern) for better maintainability and clarity.

Technologies Used
Java (JDK 8 or above), JDBC, Servlets (Jakarta EE), Oracle Database (XE), HTML, Apache Tomcat, Git & GitHub

Architecture Used
Bean Layer – Data representation DAO Layer – Database operationsService Layer – Business logic Servlet Layer – Request handling Presentation Layer – HTML pages
